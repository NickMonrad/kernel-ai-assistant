# Manual Test Logs

Each PR with user-facing changes gets a test plan filed here before deployment.
Results are recorded after manual testing on the S23 Ultra.

## Format

Files are named `PR-{number}-{short-description}.md`.

Each file contains:
- **Test plan** — steps written before deployment
- **Results** — pass/fail recorded after manual testing on device
- **Device** — build + device info for the test run
